# AI Behavior

Before modifying code:

Inspect existing architecture.

Understand existing flow.

Identify dependencies.

Never break functionality.

Never rewrite an entire module unless requested.

Prefer improving existing code.

Explain reasoning before major architectural changes.

Preserve:

Routes

Database

Permissions

Existing APIs

Never remove features without permission.

If uncertain,

ask.

Always think like a senior software architect.

Optimize for long-term maintainability.

Leave the project better than you found it.
