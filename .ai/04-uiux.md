# UI UX

Follow UI UX Pro Max principles.

Use:

consistent spacing

consistent typography

consistent colors

consistent shadows

consistent border radius

Use Bento layouts when appropriate.

Avoid crowded interfaces.

Prefer whitespace.

Improve visual hierarchy.

Every page should include:

Header

Breadcrumb

Actions

Content

Empty State

Loading Skeleton

Success Feedback

Error Feedback

Responsive layout.

Accessibility first.

Animations should be subtle.

Never overuse gradients.
